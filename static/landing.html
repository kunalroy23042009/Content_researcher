What data sources does it use?

YouTube Data API v3 for channel stats, video metrics, and search. Reddit via PRAW for community discussions and trends. Google Gemini (and optional Groq/OpenRouter) for AI analysis.

How accurate is the niche profiling?

The AI analyzes your channel title, description, subscriber count, recent video titles, engagement metrics, and upload frequency to generate a niche profile. The more content you have, the more accurate it is.

Can I use it without any API keys?

No — you need at minimum a YouTube Data API key and a Gemini API key. Both have free tiers. Check the .env.example file for setup instructions.

Is my data stored?

Analysis results are cached locally for 24 hours to avoid redundant API calls. No data is sent to third parties. On the SaaS version, your account data is stored securely.
